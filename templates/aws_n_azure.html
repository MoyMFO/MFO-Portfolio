Business Requirement

A greenhouse installs a new temperature regulation system that is made up of a monitoring system and an electrical cooling system. The cooling system monitors in real time and collects temperature and humidity in order to identify temperature peaks to turn on the cooling system and also to generate data analysis. Moy Projects was hired for the purpose of installing said system. A possible architecture is proposed below. The only restriction is that Moy Projects has an agreement with Azure that provides a very attractive cost for streaming services, however Moy Projects has the entire storage and analysis architecture in AWS. The other condition is that the important data are all those temperature records above 27 degrees.
